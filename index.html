⚖️ 第6章 物质的物理属性

6.1 质量的概念与单位

质量（m）：物体中所含物质的多少。它是物体的固有属性，不随形状、状态、温度、位置的改变而改变。

国际单位：千克（kg）。换算：1t=10³kg=10⁶g=10⁹mg。

案例1：一块冰化成水，质量变吗？
答：不变。状态变了但物质多少没变。同理：宇航员从地球到月球，质量也不变。
案例2："1kg棉花和1kg铁谁重？"
答：一样重（都是1kg）。但棉花的体积大（密度小），铁的体积小（密度大）。

6.2 天平的使用（重点实验）

四步法：

  1. 放：水平台，游码归零
  2. 调：调平衡螺母→指针指中央
  3. 称：左物右码，镊子夹砝码（大→小），拨游码
  4. 读：物体质量=砝码+游码刻度
⚠ 注意事项：不能超量程；砝码用镊子；潮湿/化学药品垫纸或放烧杯中称。
坑：天平调平后互换物码位置——读出的不是真实质量！左码右物时 m物 = m码 - m游。

6.3 密度——核心公式

定义：某种物质单位体积的质量。公式：ρ = m / V

单位：kg/m³ 或 g/cm³。换算：1g/cm³ = 1000 kg/m³

ρ水 = 1.0×10³ kg/m³ = 1 g/cm³

密度是物质的特性：同种物质密度相同；不同物质密度一般不同。温度升高→体积膨胀→密度减小（水在4℃时密度最大，这是个例外！）。

密度知识的三种应用：

应用公式示例
鉴别物质ρ=m/V测出m和V → 查密度表 → 判断是哪种物质
求质量m=ρV一块2m³大理石(ρ=2.7g/cm³)，质量=5.4t
求体积V=m/ρ1kg水的体积=1000cm³=1L

6.4 空心/实心判断（三种方法）

例题：铁球158g，体积30cm³，ρ铁=7.9g/cm³，判断是否空心？

方法① 比质量：m理论=7.9×30=237g > 158g → 实际质量小 → 空心

方法② 比体积：V铁=158/7.9=20cm³ < 30cm³ → 实际体积大 → 空心，空心=10cm³

方法③ 比密度：ρ实际=158/30≈5.27 < 7.9 → 空心

三种方法等效，选最快的一种。方法②还能直接求出空心体积！

6.5 密度测量实验

① 测固体密度（不溶于水）：

  1. 天平测质量 m
  2. 量筒中加水，读数 V₁
  3. 物体浸没，读数 V₂
  4. ρ = m/(V₂-V₁)
⚠ 先测质量还是先测体积？先测质量！如果先浸水再称，物体会沾水使质量偏大→密度偏大。

② 测液体密度（减液法）：

  1. 测烧杯+液体总质量 m₁
  2. 倒入量筒一部分，读体积 V
  3. 测剩余烧杯+液体质量 m₂
  4. ρ = (m₁-m₂)/V
减液法的好处：避免了烧杯残留带来的误差。

6.6 物质的其它物理属性

硬度：物质抵抗被划破的能力。铅笔上4B→2B→HB→2H→4H，越右越硬。

弹性/塑性：弹性→撤力恢复原状（弹簧）；塑性→撤力不能恢复（橡皮泥）。

导电/导热性：金属好，非金属差。

🔬 第7章 从粒子到宇宙

7.1 分子动理论（三大要点）

① 物质由大量分子组成，分子间有间隙（直径约10⁻¹⁰m）

② 分子永不停息地做无规则运动——热运动（温度越高越剧烈）

③ 分子间同时存在引力和斥力

压缩物体→斥力起作用（固体难压缩）；拉伸物体→引力起作用（钢丝难拉断）。破镜不能重圆→断裂处分子间距太大（>10倍分子直径），引力可忽略。

7.2 扩散现象——分子运动的证据

扩散：不同物质相互接触时，彼此进入对方的现象。

案例：香水瓶打开→满屋飘香（气体扩散）。盐撒进汤→整锅变咸（液体扩散）。金块和铅块紧压几年→互相渗入（固体扩散，最慢）。
结论：一切物质的分子都在不停运动。扩散速度：气体 > 液体 > 固体。
⚠ 区分扩散和机械运动：花香→扩散（分子运动）✓；尘土飞扬→机械运动（微小颗粒，不是分子）✗；柳絮飘飞→机械运动✗。

7.3 分子间的相互作用力

引力和斥力同时存在。距离太近→斥力>引力；距离适中→引力=斥力；距离太远→都趋近于零。

案例：两根铅柱端面磨平紧压，能吊起钩码——分子引力在起作用。端面不磨平则不能成功→只有少数分子靠得够近。

7.4 静电现象

知识点内容
摩擦起电摩擦使物体带电。带电体可吸引轻小物体。
正电荷与丝绸摩擦过的玻璃棒
负电荷与毛皮摩擦过的橡胶棒（口诀：玻正橡负）
电荷规律同斥异吸
⚠ 核心认知：摩擦起电不是创造电荷，只是电子从一个物体转移到另一个物体。

7.5 原子结构与宇宙

汤姆生发现电子→原子可分。卢瑟福核式模型：原子核（带正电）+核外电子（带负电）绕核高速运动。

原子核=质子（正电）+中子（不带电）。质子/中子由夸克构成。

微观尺度链：分子 > 原子 > 原子核 > 质子/中子 > 夸克

光年：光在真空中一年传播的距离。长度单位（不是时间单位！）。1光年≈9.46×10¹⁵m。
宇宙结构：地月系→太阳系→银河系→宇宙。哥白尼提出"日心说"。

💪 第8章 力

8.1 力的基本概念

定义：力是物体对物体的作用（施力物+受力物）。力的作用是相互的。

单位：牛顿（N）

作用效果：①改变运动状态（快慢/方向）②使物体形变

三要素：大小、方向、作用点。三者中任一改变，力的效果就不同。

案例：推门时力作用在门把手（远离门轴）比作用在门轴附近省力——作用点不同，效果不同。

8.2 弹力与弹簧测力计

弹性：撤力后恢复原状。塑性：撤力后不能恢复。

弹力：物体发生弹性形变时产生的力（拉力、压力、支持力都属于弹力）。

胡克定律：在弹性限度内，F = k·Δx（弹簧伸长量与拉力成正比）。

重要结论：弹簧截短→k变大（越短越硬）。长度变成原来的n倍→k变成原来的1/n。两根相同弹簧串联→k减半；并联→k加倍。
弹簧测力计使用：①观察量程分度值 ②调零 ③拉力沿轴线 ④视线与刻度垂直。

8.3 重力

产生原因：地球吸引。方向：竖直向下（重垂线原理）。

公式：G = mg，g ≈ 9.8N/kg（粗略计算可取10N/kg）。

重心：质地均匀、形状规则的物体，重心在几何中心。

案例：宇航员60kg，地球上G=600N。月球上g是地球的1/6，所以月球上G=100N，但质量还是60kg不变。
月球上能举起120kg物体→需要力=120×10/6=200N。地球上同样200N的力只能举起20kg。

8.4 摩擦力（三种）

种类条件公式/特点方向
静摩擦有趋势但未动f静=外力（≤fmax）与相对运动趋势相反
滑动摩擦发生相对滑动f=μN（只与N和μ有关！）与相对运动方向相反
滚动摩擦物体滚动远小于滑动摩擦—
⚠ 滑动摩擦力的"三无关"：与速度无关！与接触面积无关！与拉力大小无关！只与正压力和粗糙度有关。
经典题：物体侧放后匀速拉动→弹簧测力计示数不变。因为压力和粗糙度都没变→f不变。
测量方法：弹簧测力计匀速拉物体→F拉=f（二力平衡）。

8.5 增大/减小摩擦的方法

目的方法实例
增大有益摩擦增大压力/增大粗糙度刹车用力；轮胎花纹；体操镁粉
减小有害摩擦减压力/减粗糙度/变滑动为滚动/加润滑油/气垫轴承滚珠；气垫船；磁悬浮

8.6 力的示意图画法

①确定受力物体 ②画出作用点（通常在重心） ③沿力的方向画线段（长度表大小） ④画箭头表方向 ⑤标出力的符号和大小。

多个力作用在同一物体时，作用点都画在重心上。

🏃 第9章 力与运动

9.1 二力平衡

平衡状态：静止或匀速直线运动。

四个条件（缺一不可）：同体 + 等大 + 反向 + 共线。

判断方法：先判断物体在哪个方向平衡，再分析该方向受力。

案例1：吊灯静止→竖直方向重力=拉力；水平方向不受力。
案例2：汽车匀速→水平方向牵引力=阻力。

9.2 平衡力 vs 相互作用力（高频考点）

区别一对平衡力一对相互作用力
作用对象同一物体两个不同物体
举例书放桌上：G=N书压桌 ↔ 桌撑书
能否抵消能（合力为零）不能（作用在不同物体上）
判断口诀："一个物体→平衡力；两个物体→相互作用力"

9.3 牛顿第一定律（惯性定律）

一切物体在不受外力时，总保持静止或匀速直线运动。

⚠ 不能直接用实验验证（没有真正的"不受力"环境），是实验+推理得出的。

伽利略理想实验→牛顿总结→牛顿第一定律。

实验探究：小车从同一斜面同一高度滑下（控制初速度相同），在毛巾/棉布/木板表面滑行。表面越光滑→阻力越小→滑行越远。
推论：若表面绝对光滑（无阻力），小车将永远运动下去。

9.4 惯性（最容易混淆的概念）

定义：物体保持原来运动状态不变的性质。

一切物体在任何时候、任何状态下都有惯性。

⚠ 三大误区：
① 惯性不是力！不能说"惯性力"或"受到惯性作用"——正确说法是"由于惯性"。
② 惯性大小只与质量有关，与速度、形状无关！
③ "速度大惯性大"是错的——质量大惯性才大！
利用惯性防止惯性危害
跳远助跑、拍打衣服除尘、锤头松了撞击几下安全带、安全气囊、保持车距
刹车前倾的解释模板：汽车刹车前，乘客与车一起运动。刹车时乘客的脚随车停止，而上身由于惯性仍保持原来运动状态，继续向前，所以身体向前倾倒。

9.5 力与运动的关系（核心总结）

受力情况合力运动状态实例
不受力0静止或匀速直线太空中的飞船
受平衡力0静止或匀速直线吊灯、匀速汽车
受非平衡力≠0加速/减速/方向改变起跑、刹车、转弯
核心结论：力是改变运动状态的原因，不是维持运动的原因（牛顿推翻了亚里士多德的错误观点）。

9.6 传送带问题专题

一起匀速：物体与传送带相对静止→无相对运动趋势→不受摩擦力。
传送带加速：物体相对传送带有向后趋势→受向前的静摩擦（带动物体加速）。
传送带减速：物体相对传送带有向前趋势→受向后的静摩擦。

🌊 第10章 压强和浮力

10.1 压力与固体压强

压力：垂直压在物体表面上的力。方向：垂直于受力面。

压强：p = F / S。单位：帕斯卡（Pa），1Pa=1N/m²。

⚠ 压力 ≠ 重力！只有物体在水平面上静止时 F=G。斜面上F 经典易错：60kg人每只脚150cm²。站立→S=300cm²→p=20000Pa。走路→S=150cm²→p=40000Pa。面积差一倍，压强差一倍！

10.2 液体压强

公式：p = ρgh

特点：液体内部各个方向都有压强；同一深度各方向压强相等；深度越深压强越大；密度越大压强越大。

⚠ p只与ρ和h有关！与液体质量、体积、容器形状统统无关。
计算液体对容器底压力：F=pS=ρghS。
直筒容器：F=G液 ✓ ；上宽下窄：FG液。
记忆口诀：只有直筒→F=G；异形容器→F≠G！

10.3 大气压强

产生原因：空气受重力+有流动性。

证明存在：马德堡半球实验。 首次测量：托里拆利实验。

1标准大气压 = 1.013×10⁵Pa = 760mmHg ≈ 10m水柱

海拔越高→气压越低（3000m内每升10m约降100Pa）。气压越低→沸点越低（高原煮不熟饭→用高压锅）。
应用：吸盘挂钩、抽水机、注射器吸药液、吸管喝饮料。

10.4 流体压强（伯努利原理）

在流体（气体/液体）中，流速大的位置压强小。

飞机升力：机翼上表面凸→空气流速快→压强小；下表面平→流速慢→压强大→产生向上的升力。
站台安全线：火车驶过→空气流速大→压强小→外侧气压大→把人推向火车。

10.5 浮力与阿基米德原理

浮力方向：竖直向上。

阿基米德原理：F浮 = G排 = ρ液·g·V排

⚠ F浮只取决于ρ液和V排！与物体密度、形状、在液体中的深度均无关。

四种求浮力法：

方法公式何时用
压力差F浮=F上-F下知上下表面压力
称重F浮=G-F示题中出现"弹簧测力计示数"
阿基米德F浮=ρ液gV排知ρ液和V排，"体积"条件
平衡F浮=G物漂浮或悬浮

10.6 浮沉条件——核心判断

状态F浮 vs Gρ物 vs ρ液
上浮F浮>Gρ物<ρ液
下沉F浮ρ物>ρ液
悬浮F浮=Gρ物=ρ液
漂浮F浮=Gρ物<ρ液
🏆 漂浮黄金公式：ρ物/ρ液 = V排/V物。
例：木块漂浮，2/5露出→V排=3V/5→ρ木=3ρ水/5=0.6g/cm³。

10.7 浮力的应用

应用原理关键点
轮船空心法增大V排河→海：上浮（浮力不变，V排变小）
潜水艇改变自身重力进水→沉；排水→浮
密度计漂浮（F浮=G）刻度"上小下大"
气球/飞艇充密度小于空气的气体热空气、氢气、氦气
